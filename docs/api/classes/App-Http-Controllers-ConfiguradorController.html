ConfiguradorController.php : 34

Controlador del configurador de PC.

Gestiona el wizard de configuración de PC paso a paso. El estado de la configuración se almacena en la sesión del usuario bajo la clave SESSION_KEY ('configurador').

Estructura de la sesión: [ 'plataforma' => 'AM5', 'seleccion' => ['Procesadores' => 8, 'Placas base' => 12, ...], 'montaje' => 'con_montaje' | 'sin_montaje', 'montaje_producto_id' => 71 | null, ]

El wizard consta de 9 pasos:

  • Pasos 1-8: Categorías de componentes (Procesadores, Placas base, etc.)
  • Paso 9: Selección del servicio de montaje (obligatorio)

Opciones especiales del wizard:

  • producto_id = -1: Usar gráficos integrados del procesador (iGPU)
  • producto_id = 0: Disipador de caja incluido con el procesador
